Idle Relay Control (IOX-OUTPUT)

Select a vehicle with IOX-OUTPUT installed. You can pulse the relay manually, or enable automatic idle shutdown. Note: auto mode only runs while this page is open in MyGeotab.

Output index: Pulse (seconds):
Set Output index to the channel wired to the relay (e.g. 1 or 2). Pulse is how long to kill power (e.g. 15 seconds).
This must match the Exception Rule name you created for idle > 2m50s (or whatever duration you used).
