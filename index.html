Idle Relay Control

IOX-OUTPUT + IOX-ALERT control. Idle rule kills the bus and leaves it latched off until the driver presses the alert button or an admin restores power here.
Vehicle
Only devices with IOX-OUTPUT installed should be used here.
Status
Last command: None yet
Status is based on the last command sent from this panel.
Relay settings
This should match the IOX-OUTPUT channel wired to the engine kill relay.
Admin controls
These commands go directly to the relay. Use restore if a bus is stuck off.
Automation
When enabled, this tab will:
• Watch the idle rule (2m50s) and send relay ON (kill)
• Watch the IOX-ALERT rule and send relay OFF (restore)
